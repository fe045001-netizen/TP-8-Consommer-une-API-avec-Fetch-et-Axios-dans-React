# TP 8 : Consommer une API avec Fetch et Axios dans React

## Description

Ce projet React illustre la consommation d’API à l’aide de **fetch** et **Axios**.  
Il contient deux composants principaux :

- `FetchData` : récupère et affiche des articles depuis [JSONPlaceholder Posts]
- `AxiosData` : récupère et affiche des utilisateurs depuis [JSONPlaceholder Users]avec leur ville.

Chaque composant inclut un bouton **“Recharger les données”** pour relancer la requête et mettre à jour l’affichage.

## Installation

src/

│

├─ FetchData.js       # Composant Fetch API

├─ AxiosData.js       # Composant Axios

├─ App.js             # Composant principal qui affiche FetchData et AxiosData

└─ index.js           # Point d’entrée React
